# License and provenance policy

Before a release:

1. Pin every dependency and model revision.
2. Review each software and model license.
3. Record custom-license acceptance where required.
4. Generate an SPDX or CycloneDX software bill of materials.
5. Keep Gemma disabled unless its terms have been reviewed and accepted.
6. Document AGPL obligations for Redis 8, MinIO, and Grafana configurations where applicable.
